This file will contain build instructions.


To use the OpenRAVE based visualization, simply install OpenRAVE using the instructions here: http://openrave.org/docs/latest_stable/install/#install
In Ubuntu, a minimal set of OpenRAVE modules may be installed with:
$ sudo add-apt-repository ppa:openrave/release
$ sudo apt-get update
$ sudo apt-get install openrave-minimal, openrave-python-minimal, openrave0.8-dp-base, openrave0.8-dp-data, openrave0.8-dp-plugin-qtcoinrave, openrave0.8-dp-python
